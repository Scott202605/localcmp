# LocalCMP Frontend Prototype

This is a static CMP frontend prototype for the IoT connectivity management platform.

## Preview

Open `index.html` directly in a browser for local preview.

## Production Security

Production deployment must use HTTPS. The prototype includes a restrictive Content Security Policy, privacy policy page, cookie consent UI, masked operational flows, accessible focus states, validation feedback, and destructive-action confirmation.

## Implemented Areas

- Dashboard app shell
- Clickable module navigation with separate rendered pages
- SIM/eSIM operational navigation
- KPI cards and usage trend
- Supplier health
- Operation queue
- SIM status table
- SIM detail/lifecycle actions
- Package draft form with validation
- Package, usage/CDR, account, billing, batch, supplier, API/webhook, settings, and audit subpages
- Cookie consent and privacy policy
- Responsive layout
